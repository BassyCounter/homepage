Projects I've started:


Basic Music Guide

The purpose of this project was to help me practice with my Python programming skills, as well as to create a simple tool that I could use to help me with playing guitar. The project features a module that can be ran from the command line for simple use, and a GUI module that can be ran if you would like something a little more visual and interactive. The project mostly just allows the user to choose from a list of predefined music theory terms and displays information for it, which can be helpful for practicing improvisational guitar playing skiils and song creation.

Book Logger

I created this project to practice more with Python by implementing more complex data structures to mimic a book log that holds a book's title and author, as well as the time of when it was logged as started and finished. The data is saved to both a text file and a CSV file so that the user can view the book log information from outside of the program and have an opportunity to save it elsewhere, whether it be in the cloud, a USB flash drive, etc.
